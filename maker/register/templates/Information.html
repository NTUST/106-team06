台灣無人飛機應用設計學院 / Taiwan Drone Design and Application Development College

國內最專業針對無人機的相關知識傳遞及教學單位。課程內容從最基礎的飛行安全訓練，中階飛控系統軟硬體整合，競速無人機FPV訓練教學，至高階無人機電腦視覺及自動化系統設計。

  • 有興趣

【課程】Arduino 四軸飛行器實作坊，無人機硬體、無線遙控器、飛控軟體整合、飛行教學，一天學會

【6月3日班】四軸飛行器實作要學什麼？除了自己做 Arduino 無人機，本課程還同時學會自己做 Arduino 無線遙控器，當然還有飛行控制軟體。想要學技術、做空拍、收集環境資訊、協助救災、觀察危險地帶、競速…各種無人機的應用，都可以從這場實作課開始，利用開源硬體、開源軟體的力量，實現願望。

Webduino【 Web + Arduino 】

我們是一個高雄的熱血創意團隊，成員有韌體工程師、網頁前後端工程師以及視覺設計師，憑著一股對網頁技術無止盡的愛，在 2015 年，實現了真正用 Web 串接 Arduino 的完整過程，純粹撰寫 HTML 與 Javascript 的網頁語法，就能夠玩轉各式各樣的傳感器，為廣大的開發者和網頁設計師，創造了前所未有的開發模式，我們深信在不久的將來，網頁技術將引領開發者的潮流，帶來各式各樣的創意與契機！

我們每個月都會舉辦活動，與各大開發者社團也都有相關交流，在平均下來一個月至少舉辦三場活動 ( 教師研習、社團、展覽...等 )，於 2015 年更參與了台灣知名的大型活動 ( JSDC、WebConf、MOPCON、MakerFaire...等 )，我們相信參與這些大型的活動，可以從許多前輩、先進與同好身上，獲得許多寶貴的改善建議，同時也能將自己的技術貢獻給廣大的開發者們。
