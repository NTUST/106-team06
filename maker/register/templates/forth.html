【課程】Arduino 四軸飛行器實作坊，無人機硬體、無線遙控器、飛控軟體整合、飛行教學，一天學會

【6月3日班】四軸飛行器實作要學什麼？除了自己做 Arduino 無人機，本課程還同時學會自己做 Arduino 無線遙控器，當然還有飛行控制軟體。想要學技術、做空拍、收集環境資訊、協助救災、觀察危險地帶、競速…各種無人機的應用，都可以從這場實作課開始，利用開源硬體、開源軟體的力量，實現願望。

6個理由，為什麼要來上課？

1.本次課程主要使用Arduino模組進行教學，非常容易學習，無論是否是熟悉程式的學員都能夠輕易上手，模組化感測器控制原理教學。

2.翔探科技自有技術研發而來的四軸飛行器，包含原始程式碼以及電路設計都是翔探科技特別以競速無人機飛控技術為基礎，專為初學者開發的四軸飛行器電子積木套件。

3.翔探科技LDSLITE250的重量輕（低於100g) 旋翼的轉速也比一般低速很多，所以十分安全適合教育及實驗開發使用。

4.本課程提供旋翼機飛控最重要的體驗並學習調整PID，並且學會自己設計飛控程式。

5.動手組裝LDSLITE 250四旋翼機，體驗自造航空器樂趣。

6.PAW遙控器是翔探科技針對無人機教育自行開發的Arduino搖桿，讓學員充份發揮自造精神及樂趣。

課程內容

時間：2017/6/3(六) 早上 09:30~下午 17:30

使用 Arduino 模組、飛控電路板、nRF24L01 無線通訊模組打造一套專屬的四軸飛行器+無線遙控器。

名額：額滿為止

費用：早鳥價 4,500 元，包含所有硬體材料、午餐。

1.Arduino 模組(已焊在載板上)：2組

2.nRF24L01 2.4GHz 無線通訊模組：2組

3.飛控電路板：1組

4.電池：1組

5.LDSLITE 250 V2飛機骨架含馬達及旋翼套件：1組

6.Aeroprobing PAW搖桿電路套件：1組

注意事項：

1.學員需自備筆電，請注意使用權限為電腦管理員，筆電須有兩個以上 USB 插孔，不建議使用USB Hub。

2.Windows 7/8/8.1/10 作業系統皆可，Windows XP 不支援。請先關閉硬體數位簽章及移除原有Arduino IDE，至少需有1GB儲存空間，USB隨身碟/SD卡等外部儲存裝置亦可。

3.上課會使用到「客製的 Library」，目前不支援 Mac OS，如果您沒有 Windows OS 筆電的話，請勿報名。

4.請自備兩條 Micro USB 傳輸線。

5.請自備行動電源，供遙控器使用。

6.課程內容不適合國小、國中學生，請勿報名。

7.尊重教學團隊之著作權，因此謝絕全程錄影、錄音，但可拍照。
