Webduino【 Web + Arduino 】

我們是一個高雄的熱血創意團隊，成員有韌體工程師、網頁前後端工程師以及視覺設計師，憑著一股對網頁技術無止盡的愛，在 2015 年，實現了真正用 Web 串接 Arduino 的完整過程，純粹撰寫 HTML 與 Javascript 的網頁語法，就能夠玩轉各式各樣的傳感器，為廣大的開發者和網頁設計師，創造了前所未有的開發模式，我們深信在不久的將來，網頁技術將引領開發者的潮流，帶來各式各樣的創意與契機！

我們每個月都會舉辦活動，與各大開發者社團也都有相關交流，在平均下來一個月至少舉辦三場活動 ( 教師研習、社團、展覽...等 )，於 2015 年更參與了台灣知名的大型活動 ( JSDC、WebConf、MOPCON、MakerFaire...等 )，我們相信參與這些大型的活動，可以從許多前輩、先進與同好身上，獲得許多寶貴的改善建議，同時也能將自己的技術貢獻給廣大的開發者們。

特色

只做網頁就可快速上手:純粹利用 HTML、Javascript ...等網頁語法進行開發，不需要學習艱難的 C 或 C++，只要會寫網頁，就可以串起 Arduino，控制形形色色的傳感器。

支援電腦、平板、手機:由於使用網頁語法開發，可以藉由網頁控制元件或傳感器，因此不論是 Windows、Mac、Android 或 iOS 都可以順利運行，輕鬆實現跨平台的控制。

最簡單的開發模式:Webduino 的誕生，替物聯網開啟了最關鍵的一條路，讓網頁開發者可以更容易地跨入硬體的領域，讓硬體的開發者可以更簡單的使用網頁的服務。

豐富完整的線上資源:上百篇的教學文章、範例、展示影片，由淺入深的引入物聯網的核心開發領域，讓您在學習的過程享有最完整的教學資源。

人數破千的技術社團:Webduino 平均每個月舉辦 2.8 場活動，在 Facebook 更擁有上千人的技術討論社團，有任何問題都可在社團裡找到專人解答。

須具備的技術

後端工程師

- 必備技能：Git、JavaScript、NodeJS、RDBMS

- 加分技能：Network mgnt.、DevOps skills

- 履歷提供：個人資料、Github、成果作品

前端工程師

- 必備技能：Git、HTML、CSS、JavaScript、NodeJS

- 加分技能：AngularJS、ReactJS、WebComponent

- 履歷提供：個人資料、Github、成果作品

韌體 / 機構工程師

- 必備技能：C / C++、PCB Layout、HTTP、TCP/IP

- 加分技能：JavaScript / NodeJS、Arduino / esp8266、3D 建模與列印、機構設計與實作

- 履歷提供：個人資料、Github、成果作品

前端設計師

- 必備技能：Photoshop、Illustrator、HTML、CSS

- 加分技能：Git、JavaScript / JQuery、LESS、商品拍攝、平面設計、影片剪輯

- 履歷提供：個人資料、Github、成果作品
