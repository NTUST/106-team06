甚麼是創客?
你，就是一位

自造者（Maker，又譯為「創客」）概念來源於英文Maker和Hacker兩詞的綜合釋義，它是指一群酷愛科技、熱衷實踐的人群，他們以分享技術、交流思想為樂，以自造者為主體的社區（Makerspace）則成了自造者文化的載體。

自造者是一群酷愛科技、熱衷實踐的人群，他們以分享技術、交流思想為樂，以自造者為主體的社區則成了自造者文化的載體。 自造者空間（maker space）就是結合不同自自造者的外部能量，激發每個人的創造力。

創客計畫

獎金計畫

利用比賽，創造更多創客：在生活中，總是有許多大大小小的麻煩事困擾著我們，但是我們之所以為萬物之靈，便是造物者賦予我們創造力、想像力，讓我們能用雙手開創這個世界，發明許多實用並兼具美感的東西解決生活上一切問題，請以如何提升生活的便利性，設計屬於你的創意。

行動計畫

行政院推出為期三個月的vMaker行動計畫-「Maker．空總．擂台」，即日起在TAF創新基地熱烈展開！行政院長出席活動記者會時表示，自造者運動(maker movement)蔚為國際風潮，希望藉由「動手做」精神，結合現代高科技，讓學生從中體驗，進而發揮創新與創意，讓創意帶動進步與改變，目前行政院也擴大讓「自造者運動」更深入年輕人社群及校園中。

開源計畫

創客圈會願意主動分享及公開自己的成果資訊，讓一般人能夠獲得相關資訊，享受動手做的味道

資助計畫

若你喜歡創客，可以選擇贊助他們或者是主動掏錢購買它們的產品，有你的支持，創客為你帶來更好的生活。

Webduino【 Web + Arduino 】

我們是一個高雄的熱血創意團隊，成員有韌體工程師、網頁前後端工程師以及視覺設計師，憑著一股對網頁技術無止盡的愛，在 2015 年，實現了真正用 Web 串接 Arduino 的完整過程，純粹撰寫 HTML 與 Javascript 的網頁語法，就能夠玩轉各式各樣的傳感器，為廣大的開發者和網頁設計師，創造了前所未有的開發模式，我們深信在不久的將來，網頁技術將引領開發者的潮流，帶來各式各樣的創意與契機！

我們每個月都會舉辦活動，與各大開發者社團也都有相關交流，在平均下來一個月至少舉辦三場活動 ( 教師研習、社團、展覽...等 )，於 2015 年更參與了台灣知名的大型活動 ( JSDC、WebConf、MOPCON、MakerFaire...等 )，我們相信參與這些大型的活動，可以從許多前輩、先進與同好身上，獲得許多寶貴的改善建議，同時也能將自己的技術貢獻給廣大的開發者們。
